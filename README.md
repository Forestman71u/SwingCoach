# SwingCoach AI Prototype

A static GitHub Pages prototype for an AI golf swing coaching app.

## Features

- Upload or record a golf swing video
- Preview the video in the browser
- Run a mock AI swing analysis
- Show coaching feedback cards
- Track swing improvement areas
- Generate a recommended practice plan
- Save the latest mock analysis with browser local storage

## How to use on GitHub Pages

1. Create a new GitHub repository, for example `swingcoach-ai`.
2. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and open the GitHub Pages URL.

## Important note

This is a prototype. The AI coaching feedback is mocked in JavaScript. A real version could later use computer vision, pose estimation, and an AI model to analyze swing mechanics.
